
# sesli-yazilim-mimarisi
Yazılım toplantılarındaki ses kayıtlarını yapay zeka ile analiz ederek otomatik UML diyagramları ve iş listeleri oluşturan otonom bir platformdur. Sesi metne dönüştürür mimariyi kurgular ve etkileşimli olarak görselleştirir. Geliştirme süreçlerindeki manuel analiz ve dökümantasyon yükünü ortadan kaldırarak zaman tasarrufu ve hatasız tasarım sağlar.
 React + Vite Dosyaları.
